CARA INSTALASI PADA HP SISWA:
------------------------------------------------------------------------
1. Kirim file APK pilihan (saran: Xambro-arm64-v8a.apk) ke HP siswa 
   (bisa melalui WhatsApp, ShareIt, Flashdisk, atau link download).
2. Ketuk file APK tersebut untuk memulai pemasangan.
3. Jika HP memunculkan peringatan keamanan "Instal aplikasi dari sumber 
   tidak dikenal", pilih "Izinkan" atau "Buka Setelan -> Aktifkan Izinkan 
   dari Sumber Ini".
4. Tunggu hingga proses pemasangan selesai. Aplikasi siap digunakan!

------------------------------------------------------------------------
CARA PENGGUNAAN PADA LAPTOP WINDOWS (PC):
------------------------------------------------------------------------
1. Bagikan file "exambro_windows.exe" hasil compile ke laptop siswa
   (bisa dikirim melalui Flashdisk, Google Drive, atau jaringan lokal).
2. Siswa cukup klik ganda (double-click) pada file "exambro_windows.exe" 
   tersebut untuk langsung memulai ujian.
3. Aplikasi akan mengunci layar secara penuh (Full Screen) dan berada di 
   paling atas untuk mencegah siswa membuka browser lain atau mencontek.
4. Klik kanan, klik tombol escape, refresh, dan menu pintasan lainnya 
   dinonaktifkan demi keamanan tingkat tinggi.
5. Untuk keluar dari aplikasi, siswa atau proktor dapat menekan tombol 
   "✕ Keluar" di pojok kanan atas atau menekan tombol Alt + F4, lalu 
   masukkan PIN Keamanan proktor.
