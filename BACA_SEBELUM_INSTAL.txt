------------------------------------------------------------------------
CARA INSTALASI PADA HP SISWA:
------------------------------------------------------------------------
1. Kirim file APK pilihan (saran: Xambro-arm64-v8a.apk) ke HP siswa 
   (bisa melalui WhatsApp, ShareIt, Flashdisk, atau link download).
2. Ketuk file APK tersebut untuk memulai pemasangan.
3. Jika HP memunculkan peringatan keamanan "Instal aplikasi dari sumber 
   tidak dikenal", pilih "Izinkan" atau "Buka Setelan -> Aktifkan Izinkan 
   dari Sumber Ini".
4. Tunggu hingga proses pemasangan selesai. Aplikasi siap digunakan!
